# Demo Script

Target length: under 5 minutes.

1. Open the landing page and point to the persistent devnet/prototype disclosure.
2. Show TxLINE primary mode in environment/config and explain the simulated fallback is labelled.
3. Connect Phantom or Solflare on devnet.
4. Open Match Markets for the July 14, 2026 World Cup semifinal flow and show live stat markets: cards, offsides, throw-ins, shots and shots on target.
5. Create a YES/NO market with up to five flat conditions and one top-level AND/OR operator.
6. Deposit devnet test coins/test-USDC into the PDA vault from both sides and show Explorer links.
7. Lock the market after the participation deadline.
8. Submit the final proof package. If using fallback data, explicitly say it is simulated and not TxLINE.
9. Validate the outcome through the configured verifier identity, settle and claim a payout.
10. Run the force-refund path on a second market where final data is unavailable.
11. Close with the hackathon submission checklist: live URL or devnet endpoint, public repo, endpoint list, and TxLINE API feedback.
